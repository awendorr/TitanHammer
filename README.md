# TitanHammer

**TitanHammer**, standart kırma işlemlerinin aksine, tek bir vuruşla 3x3 (ve derinlemesine 3x3x3) alanlardaki blokları aynı anda kırabilen güçlü bir efsanevi çekiç eşyası ekleyen eğlenceli bir Minecraft eklentisidir. Madenciliği büyük ölçüde hızlandırır.

### 🛠️ Kullanım & Eşya Üretimi (Crafting)
Bu devasa çekiçler **Çalışma Masası (Crafting Table)** kullanılarak üretilebilir. 3 Farklı seviyesi vardır:

1. **Demir Titan Çekici:** Üst yatay sıraya 3 Demir Külçe, Orta yatay sıraya 3 Demir Kazma, Alt yatay sıraya 3 Demir Külçe dizerek üretilir.
2. **Elmas Titan Çekici:** Üst yatay sıraya 3 Elmas, Orta yatay sıraya 3 Elmas Kazma, Alt yatay sıraya 3 Elmas dizerek üretilir.
3. **Netherite Titan Çekici:** Üst yatay sıraya 3 Netherite Külçesi, Orta yatay sıraya 3 Netherite Kazma, Alt yatay sıraya 3 Netherite Külçesi dizerek üretilir.

Bu çekiçlerden birini elinize alıp bir bloğu kırdığınızda, etrafındaki devasa bir alan anında kazılır.

✅ **Sürüm Uyumluluğu:** Bu eklenti, **Minecraft 1.21.11** sürümünde bizzat test edilmiş ve kusursuz çalıştığı onaylanmıştır.
